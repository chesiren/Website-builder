Fonctionnement:
- Drag & drop un élément de la sidebar pour l'ajouter à la page.
- Double clic sur un élément pour ouvrir la fenêtre d'édition.
- Le bouton Sauvegarder à gauche enregistre la page dans le dossier Téléchargements.

Déplacement d'éléments avec le clavier:
1. Tab jusqu'à cet item. Pour le sélectionner, appuyer sur Espace.
2. Tab jusqu'à une nouvelle dropzone.
3. Appuyer sur Entrer pour confirmer le déplacement.
Pour dé-sélectionner un élément, appuyer sur Echap.
